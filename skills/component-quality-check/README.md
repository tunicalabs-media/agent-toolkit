## What This Skill Does
Component Quality Check is a focused review skill for React and Tailwind components. It helps agents evaluate accessibility, composability, styling consistency, and Next.js expectations before UI code is considered production-ready.

## Features
- Accessibility checks for labels, keyboard behavior, focus states, and semantics.
- Component API guidance for composition and maintainable variants.
- Styling review criteria for Tailwind and shadcn-style component conventions.

## Typical Use Cases
- Reviewing a new shared UI component before merging.
- Improving a component’s accessibility and keyboard support.
- Checking whether component props, state, and styling match project conventions.

- `SKILL.md` - Component review checklist and final quality gates.
