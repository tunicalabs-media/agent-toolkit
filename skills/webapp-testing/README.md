# Webapp Testing

## Download Source
- `npx skills add https://github.com/anthropics/skills --skill webapp-testing`
- `npx skills add https://github.com/github/awesome-copilot --skill webapp-testing`

## What This Skill Does
Webapp Testing provides a Playwright-oriented workflow for interacting with and validating local web applications. It supports browser automation, UI verification, screenshots, browser logs, form interactions, and frontend debugging.

## Features
- Browser automation examples for navigation, clicks, typing, and assertions.
- Verification and debugging patterns for screenshots, console logs, and errors.
- Reusable helper asset for local test workflows.

## Typical Use Cases
- Testing a local frontend after implementing UI changes.
- Capturing screenshots to inspect layout and visual regressions.
- Debugging browser console errors, missing elements, or broken interactions.

## Files Included
- `SKILL.md` - Playwright testing workflow, examples, guidelines, and limitations.
- `assets/test-helper.js` - Helper functions for browser-based tests.

## Notes
This skill assumes a local app can be served and reached by a browser automation tool.
