# Frontend UI Engineering

## Download Source
`npx skills add https://github.com/addyosmani/agent-skills --skill frontend-ui-engineering`

## What This Skill Does
Frontend UI Engineering helps agents build and refine production-quality user interfaces. It covers component architecture, state management, design-system adherence, accessibility, responsive behavior, loading states, and transition polish.

## Features
- Component architecture and state-management patterns for maintainable UI.
- Design-system guidance for spacing, typography, color, and avoiding generic visuals.
- WCAG-minded accessibility, focus, empty-state, and responsive-design checks.

## Typical Use Cases
- Building or modifying user-facing React/Next.js interfaces.
- Improving layout, interaction polish, loading states, and empty states.
- Reviewing UI work for production readiness across desktop and mobile.

## Files Included
- `SKILL.md` - Full UI engineering workflow, red flags, and verification guidance.

## Notes
This skill is broad and implementation-focused. Combine it with design-system or framework-specific skills for more specialized work.
