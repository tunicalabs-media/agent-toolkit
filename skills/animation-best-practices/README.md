# Animation Best Practices

## Download Source
`npx skills add https://github.com/millionco/react-doctor --skill animation-best-practices`

## What This Skill Does
Animation Best Practices provides practical CSS and UI animation rules for polished, responsive interfaces. It focuses on interaction feedback, hover states, tooltips, timing, easing, accessibility, and fixes for common motion issues such as flicker and shaky transitions.

## Features
- Guidance for button press feedback, hover behavior, and tooltip/popover motion.
- Timing, easing, and transform recommendations for fast UI interactions.
- Accessibility and touch-device considerations for motion-heavy components.

## Typical Use Cases
- Adding subtle transitions to buttons, menus, popovers, and tooltips.
- Debugging animation flicker, layout instability, or awkward transform origins.
- Reviewing UI motion for accessibility and responsiveness before release.

## Files Included
- `SKILL.md` - Canonical animation guidance with examples and implementation notes.

## Notes
The skill is CSS-focused and is most useful for UI interaction polish rather than full motion-design systems or timeline-based animation work.
