## What This Skill Does
Tailwind Design System documents how to build scalable design systems with Tailwind CSS v4. It focuses on CSS-first configuration, semantic design tokens, dark mode, component variants, responsive patterns, accessibility, and migration from Tailwind v3 patterns.

## Features
- Tailwind v4 token strategy using `@theme`, CSS variables, and OKLCH colors.
- Component architecture patterns with variants, sizes, states, and overrides.
- Advanced references for scalable design-system implementation.

## Typical Use Cases
- Creating or standardizing a Tailwind-based component library.
- Migrating design tokens from Tailwind v3 configuration to v4 CSS-first setup.
- Implementing accessible, responsive variants for shared UI components.

## Files Included
- `SKILL.md` - Main Tailwind v4 design-system guide.
- `references/advanced-patterns.md` - Additional patterns for more complex design-system work.

## Notes
This skill targets Tailwind CSS v4. Check the target project’s Tailwind version before applying its configuration patterns.
