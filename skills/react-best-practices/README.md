## What This Skill Does
React Best Practices is an agent-optimized rule pack for React performance and maintainability. It covers async waterfalls, bundle size, server-side performance, client data fetching, re-render optimization, rendering behavior, JavaScript performance, and advanced callback patterns.

## Features
- Rule-based guidance with bad/good examples for React and JavaScript performance.
- Compiled `AGENTS.md` for full agent consumption.
- Granular `rules/` files organized by performance area.

## Typical Use Cases
- Reviewing React code for performance regressions.
- Refactoring expensive renders, async waterfalls, or bundle-heavy imports.
- Maintaining or extending a structured rule set for agent evaluation.

## Files Included
- `SKILL.md` - Skill entry point and rule summary.
- `AGENTS.md` - Compiled long-form rule document.
- `rules/` - Individual rule files and templates.
