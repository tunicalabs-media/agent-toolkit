# Frontend Design

## Download Source
`npx skills add https://github.com/anthropics/skills --skill frontend-design`

## What This Skill Does
Frontend Design guides agents toward distinctive, production-grade web UI design. It is useful for building or improving pages, dashboards, components, artifacts, posters, and HTML/CSS layouts while avoiding generic AI-looking aesthetics.

## Features
- Design-thinking prompts for visual direction, hierarchy, and interaction quality.
- Frontend aesthetic guidance for polished, memorable interfaces.
- Broad applicability across React components, landing pages, dashboards, and static layouts.

## Typical Use Cases
- Styling a web UI so it feels custom and production-ready.
- Creating a new page or component with strong visual direction.
- Improving typography, spacing, color, composition, or interaction polish.

## Files Included
- `SKILL.md` - Design guidance and frontend aesthetic rules.
- `LICENSE.txt` - License information included with the imported skill.

## Notes
This skill focuses on design quality. Pair it with implementation-specific skills when code architecture, accessibility, or framework conventions are also in scope.
