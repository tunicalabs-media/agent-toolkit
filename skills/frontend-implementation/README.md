## What This Skill Does
Frontend Implementation defines a practical workflow for building frontend features in a Next.js 16-style project. It focuses on request classification, React Server Component boundaries, project conventions, quality checks, and clear delivery notes.

## Features
- RSC-first implementation guidance for pages, layouts, and shared components.
- Quality gate covering accessibility, responsiveness, state, loading, and error behavior.
- Handoff checklist for explaining changes and verification.

## Typical Use Cases
- Building a new page, route segment, or shared UI component.
- Refactoring frontend code while preserving server/client boundaries.
- Applying a consistent implementation checklist before finalizing UI work.
