# Find Skills

## Download Source
`npx skills add https://github.com/vercel-labs/skills --skill find-skills`

## What This Skill Does
Find Skills helps agents discover, evaluate, and install additional skills through the Skills CLI. It is designed for situations where a user wants to know whether a capability already exists as an installable skill.

## Features
- Search and evaluation workflow for finding skill candidates.
- Quality checks before recommending external skills.
- Installation-oriented guidance using the Skills CLI.

## Typical Use Cases
- Finding a skill for a specific framework, tool, or workflow.
- Comparing external skill options before installation.
- Helping a user extend an agent with a missing capability.

## Files Included
- `SKILL.md` - Discovery workflow, search tips, quality checks, and recommendation format.

## Notes
This skill may require network access when using external skill search or installation commands.
