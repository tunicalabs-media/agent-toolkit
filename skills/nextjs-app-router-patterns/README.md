# Next.js App Router Patterns

## Download Source
`npx skills add https://github.com/wshobson/agents --skill nextjs-app-router-patterns`

## What This Skill Does
Next.js App Router Patterns explains how to build with Next.js 14+ App Router, Server Components, streaming, parallel routes, and advanced data fetching. It is intended for implementation and optimization work in modern Next.js applications.

## Features
- App Router concepts for rendering modes, layouts, pages, and route conventions.
- Patterns for Server Components, streaming, Suspense, and parallel routes.
- Data-fetching examples for SSR, SSG, and modern React/Next.js boundaries.

## Typical Use Cases
- Building or migrating App Router features.
- Implementing streaming and Suspense-backed UI.
- Optimizing Server Component data flow and route architecture.

## Files Included
- `SKILL.md` - Complete App Router overview, quick-start patterns, and examples.

## Notes
This skill is broader and more tutorial-like than `next-best-practices`, which is organized as a topic reference index.
