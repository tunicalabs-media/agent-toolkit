# Code Review Excellence

## Download Source
`npx skills add https://github.com/wshobson/agents --skill code-review-excellence`

## What This Skill Does
Code Review Excellence teaches agents how to review code with engineering judgment rather than surface-level nitpicking. It focuses on correctness, maintainability, architecture fit, security, performance, test quality, and constructive feedback that helps teams improve without slowing delivery unnecessarily.

## Features
- Structured review process from context gathering through final verdict.
- Severity-labeled feedback patterns for blocking issues, suggestions, nits, and learning comments.
- Checklists for security, performance, testing, architecture, and maintainability.

## Typical Use Cases
- Reviewing pull requests or local code changes.
- Establishing code review standards for a team or repository.
- Mentoring developers with specific, actionable review comments.

## Files Included
- `SKILL.md` - Full code review workflow, feedback examples, review checklists, and summary templates.

## Notes
This is one of the most useful baseline engineering skills in the library because it improves how agents evaluate code, not just how they generate it.
