## What This Skill Does
Vercel Composition Patterns is a React architecture skill for building component APIs that scale. It emphasizes composition over prop proliferation, compound components, provider-based state, clear context interfaces, explicit variants, and React 19 API changes.

## Features
- Rules for avoiding boolean prop sprawl and designing compound components.
- State-management patterns for lifting state into providers and decoupling implementation details.
- Compiled `AGENTS.md` plus individual rule files for agent consumption and maintenance.

## Typical Use Cases
- Refactoring a configurable component into explicit variants or compound parts.
- Designing a shared component API for long-term maintainability.
- Reviewing React architecture for hidden state coupling or over-configured props.

## Files Included
- `SKILL.md` - Skill entry point and rule summary.
- `AGENTS.md` - Compiled long-form composition guide.
- `rules/` - Individual architecture, state, pattern, and React 19 rule files.
